About TweetApp

Our Goals

This website provides a group of users to post to Twitter using a single account. On this website, users are able to submit a message to be potentially posted to Twitter. This message is reviewed by the community and has to be approved by a certain amount of votes. After this threshold is met, the message will be posted on twitter via a registered account for the group.
